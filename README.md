# Learn-CP-PSC

Learning workspace for our PSC: Knowledge tracing assistant for learning
programming

## Repository Overview

This repository contains multiple subprojects and experiments:

- `AKT/` - AKT (Attentive Knowledge Tracing) implementation using Pytorch
- `CF-Tracker/` - Codeforces recommender web app (Django-based)
- `Dynamic-cognitive-diagnosis/` - MCMC-based method to estimate central element in poset
- `Prerequisite-graph/` - prerequisite graph generation and visualization

## Quick Start

### 1) Clone the repository

```bash
git clone <your-repo-url>
cd Learn-CP-PSC
```

### 2) Choose a subproject

Each folder is currently managed independently. Enter the folder you want to run and follow its local requirements.
